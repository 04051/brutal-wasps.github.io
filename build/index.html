About the scene

At Work

The Global Pandemic caused due to Covid Outbreak has confined us all to our homes. Since all of our activites are now being carried out remotely, which also includes the the work, we thought a portrayal of our messy work desks these days would be relatable to everyone.
"At Work" is therefore a ray-traced concept that sums up our situations perfectly.

Features

The Following Features were implemented as a part of our Ray Tracer

  • Acceleration Structures

    Regular grid acceleration structure was implemented in our ray tracer on multiple levels. On upper level, objects are divided into cells, and then primitives inside objects are divided into separate grids

  • Ray Tracer

    A hierarchy was created for ray tracers which include a basic ray tracer and a whitted ray tracer, which allows support for reflected rays.

  • BRDFs

    BRDFs are implemented for several types of materials, which include, Lambertian, Glossy Specular, Perfect Specular.

  • Primitives

    Some basic primitives supported by our ray tracer are Planes, Triangles and Spheres.

  • Materials

    A material heirarchy was established and several classes for material were created. The classes include support for matte, glossy, reflective and transparent

  • Camera

    Both Parallel and Perspective views were implemented

Bonus Features

The Following additional Features were implemented as a part of our Ray Tracer

  • Lighting

    Three types of light were implemented in our structure. Ambient, Point, and Directional light

  • Anti-Aliasing

    A hierarchy was created for anti aliasing which supports jitted and Regular sampling

  • Reflected Ray

    The whitted ray tracer supports reflected rays upto a specified level of recursion.

  • Shadows

    Shadows are supported by our ray tracer which are casted by Point Light sources giving a realistic feel to the renders

  • OBJ Reader

    An obj file reader is implemented which reads OBJ files with triangular meshes and creates objects inside our ray tracer

Build Files

The following build files were used for creating different images and these available in the build folder

  • Acceleration

    The build file for acceleration can be found in raytracer\build\accelerationBuild.cpp.
    Acceleration can be toggled by changing the boolean variable ACCELERATION at line 29. Variables step and step2 define the number of primitives being used in the image

  • Shadows

    The build file for Shadows is raytracer\build\shadowExample.cpp. The shading can be toggled by changing the variable SHADOW to true/false.

  • Light

    The build file for Light is test2.cpp. Each of the lights can be removed by commenting out the section for relevant light.

  • Work Desk

    The build file for the final workdesk is final_scene.cpp. We have around 200k primitives.

  • Reflected Ray

    The build file for reflected ray is test_reflective.cpp. The depth has been set to 5. The tracer can be changed by altering line 50, Whitted tracer would compute the reflected rays while the simple tracer will not.

  • BRDFs

    Reflective Material: Build file same as Reflected Ray
    Transparent Material: Build file: test_reflective2.cpp
    Glossy and Matte: Build file: same as lighting

  • Anti-Aliasing

    The build file for the view above is gummyBear.cpp. The anti aliasing feature can be turned on by changing the sampler on line 55.
